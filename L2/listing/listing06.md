Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передачи их в качестве аргументов функции.

```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
[3 2 3]

Слайс представляет из себя структуру с тремя полями: указатель на массив, длина и емкость. При передаче слайса в функцию, происходит поверхностное копирование, то есть в функцию передается указатель на массив, длина и емкость.
Если внутри функции мы изменяем поля структуры, то это не отражается на оригинале. Но если мы будем менять элементы массива, то эти данные будут видны и на оригинале, но лишь в рамках его оригинального размера и до момента превышения capacity.

1) Изначальный массив имеет size=3, capacity=3.
2) После передачи его в modifySlice его структура была скопирована побайтно в локальную область видимости функции.
3) После этого мы изменили первый элемент массива на "3", это отразилось на оригинале, так как указатель у оригинала и у локальной копии совпадают.
4) Мы вызвали функцию append, в попытке добавить элемент "4" в конец массива. Прежнего Capacity не хватало, потому был выделен новый участок памяти размера 6, в него были скопированиы данные по старому указателю, а затем добавлен новый элемент. Очевидно, эти изменения не коснулись оригинала из-за сменившегося указателя на данные.
```
